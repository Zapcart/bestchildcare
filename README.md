## 🚀 Quick Start

1. **Add Images**: Place the following images in the root directory:
   - `childpic1.png` through `childpic5.png` (for hero slider)
   - `favicon.png` (website icon)
   - `logo.png` (for structured data)

2. **Configure AdSense**:
   - Replace `ca-pub-XXXXXXXXXXXXXXXX` with your actual AdSense Publisher ID in `index.html`
   - Replace `1234567890`, `1234567891`, `1234567892` with your actual Ad Slot IDs
   - The AdSense script is already included and will load automatically
   - Ad placements are policy-compliant and positioned between content sections

3. **Set Up Payment Gateway**:
   - In `script.js`, update the `proceed-donation` button handler to redirect to your payment gateway
   - Replace the alert with actual payment gateway integration

4. **Update Contact Email**:
   - Replace `hello@helpcharityfoundation.org` with your actual email address throughout the site

5. **Deploy**:
   - Upload all files to your web hosting
   - Ensure HTTPS is enabled (required for AdSense)
   - Test all functionality
